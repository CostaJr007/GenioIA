# GenioIA — AI-Powered Study Assistant

Mobile app built with React Native and Expo that helps students learn faster using AI. Chat with an AI tutor, generate flashcards, record and transcribe lectures, and get instant summaries.

## Features

- **AI Chat** — Chat with an AI tutor powered by OpenAI
- **Flashcards** — Auto-generate flashcards from any content
- **Lecture Recorder** — Record lectures and get AI transcriptions
- **OCR Scanner** — Scan printed materials and extract text
- **Summaries** — Generate concise summaries from long texts
- **Thesis Assistant** — Get help structuring and writing your thesis
- **Auth** — Secure login with Encrypted Storage

## Tech Stack

- **Framework:** React Native 0.81, Expo SDK 54
- **Language:** TypeScript
- **Navigation:** React Navigation (bottom tabs + native stack)
- **AI:** OpenAI API
- **Media:** expo-av, expo-camera, expo-image-picker, expo-media-library
- **Storage:** expo-file-system, expo-secure-store, expo-clipboard
- **UI:** Lucide React Native icons, react-native-svg, react-native-markdown-display

## Getting Started

```bash
# Install dependencies
npm install

# Start Expo development server
npx expo start

# Run on Android
npx expo run:android

# Run on iOS
npx expo run:ios
```
